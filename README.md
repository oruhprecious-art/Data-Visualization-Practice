# Overview

This project demonstrates how to create a simple bar chart in Python using the Matplotlib library. The chart visualizes categorical data with corresponding numerical values and saves the output as an image file.

# Requirements

Make sure you have Python installed along with the following libraries:

matplotlib

numpy

You can install them using:

pip install matplotlib numpy

# Description of the Code

The script performs the following steps:

Imports the required libraries (matplotlib.pyplot and numpy)

Defines categories and their corresponding values

Creates a bar chart

Adds a title and axis labels

Saves the plot as an image file

Displays the plot on the screen

# Output

The program generates a bar chart.

The chart is saved as bar_example.png in the project directory.

# How to Run

Save the script as bar_plot.py

Open a terminal or command prompt

Run:

python bar_plot.py

# Use Case

This example is useful for beginners learning:

Data visualization

Matplotlib basics

Python plotting workflows
